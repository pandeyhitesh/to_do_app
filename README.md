# ToDo
### To do app

## Features
#### - Add new ToDo
##### - ToDo Title
##### - Description
##### - Task Date and Time
##### - ToDo Importance 
#### - Delete a perticular ToDo
#### - Clear all ToDo

#### - Does not require internet connection
